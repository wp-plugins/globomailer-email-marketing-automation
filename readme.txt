== Installation ==

1.	In your WordPress admin panel, go to Plugins > New Plugin, search for "GloboMailerEMA" and click "Install now". Alternatively, download the plugin and upload the contents of globomailerema.zip to your plugins directory, which may be  /wp-content/plugins/. 
2.	Activate the GloboMailerEMA plugin through the 'Plugins' menu in WordPress. 
3.	A tab « GloboMailerEMA » must appear in your wordpress side navigation panel, then set your <a href="https://apps.globomailer.com/customer/api-keys/index?pk_campaign=wordpress_plugin&pk_kwd=api-keys" target="_blank">GloboMailer API keys</a> in the plugin homepage.

To get GloboMailer API key, you have to <a href="https://apps.globomailer.com/customer/guest/register?pk_campaign=wordpress_plugin&pk_kwd=create-account" target="_blank">create an account</a>. It's free and takes less than 2 minutes!

== Frequently Asked Questions ==

= What is GloboMailer? =
Easy to use, Low Cost, Email Marketing. Powerful email marketing web service that is extremely easy to use. You can Manage your subscribers, send your campaigns and track results.

= Why use GloboMailer as a my SMTP relay for my website? =
By using GloboMailer’s SMTP, you will avoid the risk of having your legitimate emails ending up in the spam folder and you will have statistics on emails sent : deliverability, opens, clicks, etc. GloboMailer’s proprietary infrastructure which we fully operate and manage optimizes your deliverability and let you focus on what you know best.

= Why do I need a GloboMailer account? =
GloboMailer for WP plugin uses GloboMailer’s API to synchronize lists, contacts, website forms, send emails. Creating an account with GloboMailer is free and takes less than 2 minutes. Once logged in your contact, you can get the API key from this page.
 
= Do I have to pay to use the plugin and send emails? =
No, the plugin is totally free and you can test the plugin with our free plan which gives you 500 emails free. If you need to send more than 500 emails, we invite you to see our pricing. For example, the Micro plan is $7.44 / month and allows you to send up to 40,000 emails per month. All GloboMailer plans are without any commitment.

= How can I get support? =
If you need some assistance, you can post an issue in the tab « Support » or send us an email on support@globomailer.com.

= How do I create a signup form? =
In order to create a signup form, you have to :
1. Go to Wp admin > GloboMailerEMA> Settings in order to define your form’s fields and settings
2. Integrate the form in a sidebar using a widget from WP panel > Appearance > Widgets. GloboMailerEMA widget form should appear in your widgets list, you just to have to drag and drop the widget into the sidebar of your choice.
